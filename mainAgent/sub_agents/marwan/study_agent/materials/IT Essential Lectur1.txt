1.1 Personal Computer
--------------------------------------------------

Computer Definition:
A computer can be defined as:

1) A person who makes calculations, especially with a calculating machine.
2) A programmable device that can store, retrieve, and process data.
3) An electronic device that can retrieve, store, process, and output data in binary form according to instructions in a program.
4) A programmable electronic device designed to accept data, perform mathematical and logical operations, store data, and display results.

Examples of computers:
Mainframes, desktop computers, laptops, tablets, smartphones, smartwatches, and IoT devices.

--------------------------------------------------
Basic Personal Computer System
--------------------------------------------------

A computer system consists of:
• Hardware
• Software

Hardware:
Physical components such as case, storage drives, keyboard, monitor, cables, speakers, and printers.

Software:
• Operating System (controls computer operation)
• Applications (perform specific tasks)

--------------------------------------------------
1.2 PC Components
--------------------------------------------------

Case:
• Houses internal components such as PSU, motherboard, CPU, RAM, drives, and adapter cards.
• Types: Super tower, full tower, mid tower, mini tower, cube case.

Quiz Answer:
Two important factors when choosing a case:
• Number of internal drive locations.
• Size of the motherboard and power supply.

--------------------------------------------------
Power Supply (PSU)
--------------------------------------------------

• Provides power to all components.
• Converts AC to low-voltage DC.
• Measured in Watts.
• Must support current and future upgrades.
• Uses input voltage selector.
• Ohm’s Law: W = V × A

Common Connectors:
• 20/24-pin main power connector
• SATA connector
• Molex connector
• Berg connector
• 4/8-pin CPU connector (ATX12V)
• 6/8-pin PCIe connector

Common Voltages:
• 3.3V
• 5V
• 12V

Rails:
Single rail, dual rail, multi rail (Modular PSU)

--------------------------------------------------
Motherboard
--------------------------------------------------

• Main printed circuit board.
• Connects all components.
• Contains buses for data transfer.
• Includes CPU socket, RAM slots, expansion slots, chipset, BIOS/UEFI, SATA connectors, USB headers.

Main Power Connector:
20 or 24-pin.

--------------------------------------------------
Chipset
--------------------------------------------------

Controls communication between:
• CPU
• RAM
• Expansion slots
• Storage devices

Northbridge:
• RAM
• Video card
• CPU communication

Southbridge:
• I/O devices
• Hard drives
• USB ports

--------------------------------------------------
Motherboard Form Factors
--------------------------------------------------

• ATX
• Micro-ATX
• ITX

Form factor determines:
• Case type
• Power supply type
• Component compatibility

--------------------------------------------------
CPU (Central Processing Unit)
--------------------------------------------------

• Brain of the computer.
• Executes instructions.
• Measured in MHz or GHz.

CPU Architectures:
• PGA
• LGA
• BGA

Multi-Core CPUs:
• Dual Core
• Triple Core
• Quad Core
• Hexa Core
• Octa Core
• 16-core and more

Technologies:
• Hyper-Threading
• HyperTransport
